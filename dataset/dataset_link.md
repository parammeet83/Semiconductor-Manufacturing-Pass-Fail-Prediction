# Dataset Information

## Dataset Name

SECOM (Semiconductor Manufacturing Process Dataset)

## Dataset Used

signal-data.csv

## Description

The dataset contains sensor measurements collected from a semiconductor manufacturing process.

It is used to predict whether a manufactured semiconductor product passes or fails the quality inspection.

## Features

- Total Samples: 1567
- Total Features: 592
- Target Variable: Pass/Fail

## Source

UCI Machine Learning Repository

https://archive.ics.uci.edu/dataset/179/secom
